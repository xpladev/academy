트랜잭션 생성하기

XPLA 테스트넷 토큰을 이용하여 블록체인에 데이터를 기록해봅시다.

📄️ XPLA 전송 트랜잭션 만들기

블록체인에 기록된 데이터들을 트랜잭션(Transaction)이라고 부릅니다. 예를 들어 Alice가 Bob에게 1 XPLA를 보냄이라는 트랜잭션을 만들어 블록체인에 기록할 수 있겠군요.

📄️ Explorer에서 트랜잭션 확인하기

XPLA Explorer를 이용하면 트랜잭션 데이터를 자세히 살펴볼 수 있습니다. 이전 단계에서 만든 트랜잭션 데이터를 다음과 같은 절차로 확인해봅시다.

📄️ 코드로 트랜잭션 데이터 가져오기

이번 단계에서는 코드를 작성하여 이전 단계에서 만든 트랜잭션 데이터를 직접 가져와봅시다.

📄️ 심화 - xplad로 트랜잭션 생성 및 데이터 가져오기

xplad로 cli에서 트랜잭션을 생성하고, query를 보낼 수도 있습니다. 이전 단계에서 xplad를 설치하지 않으셨다면, 다음 단계로 넘어가셔도 좋습니다.
