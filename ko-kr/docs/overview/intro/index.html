|- Settings

XPLA Academy에서 사용한 모든 예제 코드는 XPLA Github에서 확인하실 수 있습니다. XPLA Academy를 통해 여러분의 Web3 게임 개발을 위한 지식을 습득해보세요. 그럼 XPLA 블록체인에서 사용할 지갑부터 만들어봅시다.

XPLA 블록체인의 특징은 무엇인가요?

XPLA 블록체인은 오픈 소스이며, Tendermint Tool 기반의 Proof-of-Stake(PoS) 블록체인입니다. 또한 Cosmos SDK를 사용하면서, 동시에 EVM(Ethereum Virtual Machine) 호환성도 갖추고 있어서 Rust와 Solidity 언어로 컨트랙트를 작성할 수도 있습니다.. 더 자세한 정보는 XPLA Docs를 참고해보세요.

아직 위 내용을 모두 이해하지 못해도 괜찮습니다. XPLA Academy를 통해 관련 지식을 배우게 될 것입니다.

XPLA 블록체인에 관한 다른 문서가 있을까요?

XPLA 블록체인과 관련된 문서는 XPLA Academy 사이트 맨 하단에서 Dev Resources를 참고해보세요.
