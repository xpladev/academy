## Creating a Wallet {#createwallet}

Create a Wallet in just a few steps using Vault:

1. [Select "New wallet"](/docs/settings/create-wallet/create-wallet-with-vault#clicknewwallet)

2. [Set up wallet info and securely store the mnemonic](/docs/settings/create-wallet/create-wallet-with-vault#savemnemonic)

3. [Verify the mnemonic](/docs/settings/create-wallet/create-wallet-with-vault#verifymnemonic)

4. [Connect the created wallet to Vault](/docs/settings/create-wallet/create-wallet-with-vault#connectwallet)

### 1. Select "New wallet" {#clicknewwallet}

Launch Vault and select `New wallet`.

### 2. Set up wallet info and securely store the mnemonic {#savemnemonic}

Fill in the wallet's desired name and password, and SECURELY STORE the mnemonic words. (Always keep it safe!!!!)

:::caution

The mnemonic is crucial for wallet recovery. That means you need to remember your mnemonic to get back into your account. 

If anyone knows or steals your Wallet mnemonic, it could lead to a complete loss of your wallet assets, which we definitely want to prevent. Therefore, keep it in a safe place and never disclose it to anyone. (Not even to your favorite cat.) 

:::

Then, hit the `Submit` button.
