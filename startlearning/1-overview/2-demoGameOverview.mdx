# Demo Game Overview

XPLA ACADEMY는 [Web3 게임인 Break The Bricks](/playgame)와 [Web3 Gaming Ops 페이지](/ops)를 [오픈소스](https://github.com/xpladev/academy)로 제공합니다.
[Break The Bricks 게임](/playgame)에는 START HERE mode와 ADVANCED mode가 있습니다.
먼저 START HERE mode가 어떻게 구성되어 있는지 살펴봅시다.

<GameArchitectureTutorial />

<br/>

START HERE mode는 지갑 생성, 토큰 수령, NFT 민팅 등 Web3 기본 기능을 XPLA ACADEMY의 [Start Leaning](/startlearning/overview/intro)과 연결하여 학습할 수 있습니다. 
START HERE mode의 모든 스테이지를 완료하면, ADVANCED mode에서 사용하게 될 계정을 만들 수 있습니다.
이 계정은 START HERE mode의 STAGE 1에서 만든 지갑과 연결되어 데이터베이스에 저장됩니다.

START HERE mode를 통해 게임 계정을 만들면, ADVANCED mode를 즐길 수 있습니다. ADVANCED mode가 어떻게 구성되어 있는지 살펴봅시다. 
ADVANCED mode는 본격적인 게임 서비스이며, [Web3 Gaming Ops](/ops)와 연결되어 토큰, NFT, 최고점수 블록체인 기록 등의 Web3 게임 운영을 체험할 수 있습니다. 

<GameArchitectureAdvanced />

<br/>

ADVANCED mode를 통해 게임 재화인 DIAMOND(<DIAMOND />)를 얻을 수 있습니다.
[Web3 Gaming Ops](/ops)에서는 DIAMOND(<DIAMOND />)를 CW20 토큰인 ACADEMTY-TKN(<ACADEMYTKN />)으로 [Convert](/startlearning/tutorial/deep-understand-xpla/convert)할 수도 있고, ACADEMY-TKN(<ACADEMYTKN />)과 Testnet XPLA(<TESTNETXPLA />)를 [Swap](/startlearning/tutorial/deep-understand-xpla/swap)할 수도 있습니다.
유저는 NFT Shop에서 ACADEMY-TKN(<ACADEMYTKN />)을 지불하여 게임 장비인 PADDLE NFT를 민팅할 수 있고, PADDLE NFT를 게임 내에서 이용해서 더 높은 점수를 기록할 수 있습니다.

[Web3 Gaming Ops](/ops)의 Record, Convert, Mint, Swap 4 가지 Web3 기능을 통해 게임과 블록체인을 연결해주었습니다. 여러분도 XPLA ACADEMY를 참고하여 직접 만든 게임과 XPLA 블록체인을 연결해보세요!
